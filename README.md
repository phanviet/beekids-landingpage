# Technical Stack

This project is static website, use `parcel` to build and run:
- HTML5
- SASS
- Bootstrap 4

# Development

If you don't have SASS, then please run this first: `npm install -D sass`

Install dependencies, run `npm install`

Run project `npm run dev`

# Build

Run `npm run build`
